# GigFlow - Smart Leads Dashboard

Full-stack lead management dashboard built using MERN stack.

## Tech Stack

### Frontend
- React
- TypeScript
- TailwindCSS

### Backend
- Node.js
- Express.js
- MongoDB
- TypeScript

## Features

- JWT Authentication
- Role Based Access Control
- Lead CRUD
- Filtering
- Search
- Sorting
- Pagination

## Setup

### Backend

```bash
cd server
npm install
npm run dev

Create a `.env` file using `.env.example`
